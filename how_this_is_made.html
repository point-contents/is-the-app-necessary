The Makefile shows what is happening

The top of the page contains a blurb, this is created using pandoc to convert blurb.md to html

At the bottom of the page there is an index.

This is created with a tree and sed command that creates a tree of the directory and saves this as a html section


The python script generate_html creates an html table from products.csv

generate_html also contains placeholder html which gives the page its structure.

generate_html then uses all of the previously mentioned htmls and stiches them together to make the page that is index.html


This is all done using the make command.

Netlify is watching for pushes to the main branch of the respository.

If there is a push, Netlify will run the make command publish the directory, serving index.html statically.
